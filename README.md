# Courses App

### Required
- [NodeJS](https://nodejs.org/en/)

### Install
1. Clone or download this repository;
2. `yarn` or `npm install` in root directory;
3. `yarn dev` or `npm run dev` for start server;
4. Open `localhost:3000` in your browser;
